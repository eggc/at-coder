切り取り開始位置を少しずつずらしていけば全部作れる。
その種類は N-M+1種類。

連続部分列の項を重みつけながら足した和が最大になるように計算する。
この計算は O(M) かかる。

これが最大になるように探していたら
O(NM) かかるだろう

おそらく効率化しなければいけない
