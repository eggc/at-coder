* 例題解いてみる

S = abcbac

** i=1 のとき

0 < l <= 5 の範囲で探索する。
S[k] != S[k+1] は全ての k で満たされている。なので l = 5 となる。

** i=2 のときの l の決定方法

0 < l <= 4 の範囲で探索する

2文字ずらして一致を調べる。
S[k] != S[k+2] は S[2] == S[4] で不一致となる。なので l = 1 で終わり。
